Server Commands
Command Description
end_round cause current round to automatically draw
map_restart reset the current map and player scores
forceteam directly set the team of a certain player (1 for USA, 2 for UPKO) (syntax: forceteam <player> <team>)
shuffle_teams evenly shuffle all active players between the two teams
stfu globally silence spammy players say text
unstfu unsilence a previously silenced player
nextmap automatically enter intermission and load the next map in the mapcycle
cancelvote cancel any vote in progress
passvote pass any vote in progress
sv_kill kill the specified player (syntax: sv_kill <player>)
dc_changelevel enter intermission then change the level
 
Vote Commands
Command Description
Use the command "callvote" followed by a valid vote command below to start a vote (performed in the console).  example "callvote swap_teams"
map_restart Restart a map.
nextmap Skip the current map and go to the next one in the rotation.
swap_teams swap the teams and point then reset the round
shuffle_teams mix up the teams then reset the round
map <mapname> Change the map (example: "callvote map dc_countryside")
kick <player> Kick a player for 10 minutes
stfu <player> silence a player
unstfu <player> unsilence a player
sv_gravity <value> Change the gravity
* Voting can be enabled/disabled with mp_allowvote (0/1)
* Admins can lock out certain votes (mp_lockvotes)
