Server Cvars
Server Cvars

Name Default Value Description
mp_fraglimit 0 The number of points a player can achieve before the next map in the mapcycle will automatically load. (0 for none)
mp_timelimit 0 The number of minutes between before the next map in the mapcycle will automatically load. (0 for none)
mp_pointlimit 0 The number of points a team can achieve before the next map in the mapcycle will automatically load. (0 for none)
mp_roundlimit 0 The number of rounds played before the next map in the mapcycle will automatically load. (0 for none)
mp_friendlyfire 0 Controls whether teammates can damage each other.
mp_falldamage 1 1 for the standard progressive falling, 0 for a fixed 10 points of damage.
mp_forcerespawn 1 Automatically respawns players when their respawn time is up.
mp_spawninvul 5 The number of seconds a player is invincible after respawning.
mp_minplayers 0 The minimum number of players ready before the first round will start.
mp_warmup 30 The number of seconds before the first round will start.
mp_chattime 10 The number of seconds after the intermission starts that players are allowed to talk to each other.
sv_corpsetime 15 The number of seconds that corpses will lie around on the battlefield.
mp_freespectator 1 Enable/Disable free floating spectators.
mp_notifyunbalanced 15 The number of seconds before the server will notify that the teams are unbalanced (0 for no notification).
mp_minswitchtime 10 The minimum number of seconds a player has to wait to switch teams.
mp_forcebalance 0 Force players to join the team with the fewer number of players.
mp_maxspam 10 The max amount of spam before a player's text is blocked. Spam values decrease at one unit per second.
mp_sayspam 3 The amount of spam a saytext line generates.
mp_nickspam 5 The amount of spam changing names generates.
mp_nicknotify 1 Enable/Disable global notification of name changes.
mp_allowvote 1 Enable/Disable Desert Crisis' built-in voting.
mp_lockvotes NONE Disable different client vote commands. This is a space delineated list. (Example: mp_lockvotes "sv_secrets map kick" will disable voting for secrets, changing the map, and kicking players.)
mp_minvotewait 60 The minimum amount of time between successive callvotes from a single client.
mp_weapondrop 1 Enable/Disable the dropping of weapons when a player dies. 2 to cause players to drop all weapons on death.
mp_lives 0 Override a maps default maximum number of lives per player. (0 for no change)
